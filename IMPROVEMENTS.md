# 改善案メモ（2026-06-02 ローカルテスト後）

## 優先度：高

### 1. マークダウン太文字の未レンダリング
- **現象**：アドバイス本文中の `**テキスト**` がそのまま `**テキスト**` と表示される
- **原因**：`app.js` のアドバイス表示部分でHTMLエスケープのみ行いマークダウン変換していない
- **対応案**：表示前に `**...**` → `<strong>...</strong>` に変換する処理を追加（簡易パース）
  - ライブラリ不要、正規表現で対応可能：`text.replace(/\*\*(.+?)\*\*/g, '<strong>$1</strong>')`

## 優先度：中

### 2. アドバイス評価のページ分割
- **現象**：アドバイス1・2・3の評価が1ページに縦に並んでいる
- **問題**：次のアドバイスがすぐ見えてしまい、比較バイアスが生じやすい／操作しにくい
- **対応案**：評価ステップを3分割する
  - 現在：`4/6 評価`（1ページで3件表示）
  - 変更後：`4/6 アドバイス1評価` → `5/6 アドバイス2評価` → `6/6 アドバイス3評価` → `7/7 最良選択` → `完了`
  - 進捗バーのステップ数も合わせて調整

## 優先度：低（公開前に対応）

### 3. イントロ・同意文の本文記入
- **現象**：「ここに正式な実験説明、同意文言、謝礼、問い合わせ先を記載してください」というプレースホルダが表示されている
- **対応**：正式な実験説明文・同意文・謝礼額・問い合わせ先メールを記載する
